Import time series data from mysql to InfluxDB.

Installation
============

If you are on Debian/Ubuntu, you can install the dependences with this command::

    $ sudo apt-get install python-influxdb
    $ sudo apt-get install python-mysqldb

How to use
============

As this project is now a phototype version, a lot of things is there about to do.
If you want to try this phototype, please edit the executable file `run` in order to connect to your mysql/InfluxDB.
